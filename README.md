# 🧠 App de Aprendizaje de Lengua Kogui con IA

Aplicación educativa tipo Duolingo diseñada para el aprendizaje de la lengua kogui, desarrollada con apoyo de inteligencia artificial y pensada para funcionar en entornos con conectividad limitada (offline/híbrido).

## 🌎 Objetivo

Preservar y difundir la lengua kogui mediante una herramienta digital accesible, interactiva y culturalmente respetuosa, que permita aprender vocabulario, pronunciación y estructuras básicas del idioma.

## ✨ Características principales (Prototipo)

- 📚 Lecciones interactivas de vocabulario y frases
- 🔊 Audio de pronunciación nativa
- 🧠 Ejercicios adaptativos con IA
- 📱 Funcionamiento offline con sincronización posterior
- 🌐 Modo híbrido (online/offline)
- 🏆 Sistema básico de progreso
- 🗂️ Contenido cultural contextualizado

## 🤖 Uso de Inteligencia Artificial

El sistema utilizará modelos de IA para:

- Generación de ejercicios personalizados
- Reconocimiento o evaluación de respuestas
- Posible procesamiento de voz (futuro)
- Adaptación al ritmo del usuario

## 📡 Modo Offline

El prototipo está diseñado para:

- Descargar paquetes de lecciones
- Guardar progreso localmente
- Sincronizar datos cuando haya conexión

## 🏗️ Tecnologías (tentativas)

- Backend: Serverless / API REST
